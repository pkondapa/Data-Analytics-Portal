DESCRIPTIVE STATISTICS INFORMATION

Dataset Information

Number of variables: Displays the total number of variables/columns in your dataset

Number of records: Displays the total number of records/observations(rows) in your dataset

Total missing values(%): Gives the percentage of missing values in the dataset which is given by
(total number of values - total number of non-null values)/(total number of values)

Variables

For each variable in the dataset, it displays the type of the variable -- Numeric, Date, Text, Categorical and the percentage of missing values. If the variable is numeric, it displays the below statistical information:

1. Minimum Value: The minimum value in the column

2. Maximum Value: The maximum value in the column

3. Mean: The average of the values in the column given by (sum_of_values)/(number_of_values)

4. Median: The "middle" value in the list of numbers (values in the column arranged in an ascending order)

5. Standard deviation: A measure that is used to quantify the amount of variation or dispersion of a set of data values. A low standard deviation indicates that the data points tend to be close to the mean (also called the expected value) of the set, while a high standard deviation indicates that the data points are spread out over a wider range of values.

6. Quantile Statistics: The three cut points that will divide a dataset into four equal-size groups

If the variable is not numeric, it displays the below variable information:

1. Count: The number of non-null values in the column

2. Unique: The number of unique values for the variable

3. Top: The most frequent value in the column

4. Frequency: Number of occurences of the most frequent value in the column

Sample Data

It displays the first 5 rows in the dataset for a basic overview of the data.
